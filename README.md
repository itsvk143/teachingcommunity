# Teaching Community

A comprehensive platform connecting Teachers, Coaching Institutes, Schools, Parents, and Students. Note: This project was formerly known as "Next-Auth".

## Features
- **Profiles**: Dedicated profiles for Teachers, Non-Teaching Staff, Coaching Institutes, Schools, Parents, and Students.
- **Home Tuition**: Connect parents/students with home tutors.
- **Admin Panel**: Robust administration for managing all entities.
- **Search & Filter**: Public directories for finding educators and staff.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure
- `src/app`: App Router pages and API routes.
- `src/components`: Reusable UI components.
- `src/model`: Mongoose models for Database.
- `src/lib`: Utility functions and DB connection.

## Deploy
Deploy easily on Vercel or any Node.js hosting.
